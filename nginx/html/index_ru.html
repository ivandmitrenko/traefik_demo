Работающие сервисы

Docker

Docker это открытая платформа для разработки, доставки и запуска приложений.

Traefik

Traefik это маршрутизатор с открытым исходным кодом, легко и просто публикующий ваши приложения.

Oracle cloud

Облачная инфраструктура Oracle обеспечивает эластичность и эффективность публичного облака.

Nginx

NGINX - это бесплатный, с открытым исходным кодом, высокопроизводительный HTTP-сервер и обратный прокси-сервер, а также IMAP/POP3 прокси-сервер.

Git

Git - это бесплатная распределённая система управления версиями с открытым исходным кодом, предназначенная для быстрой и эффективной работы с проектами.

База данных

Система управления базами данных (СУБД) - это программный комплекс, предназначенный для определения, манипулирования, извлечения и управления данными в БД.
